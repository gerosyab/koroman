# 한국어->로마자 변환 서비스의 구현 지침

한국어의 로마자 변환 표기는 KOROMAN 오픈소스를 기반으로 최신의 표준 국어의 로마자 표기법 [문화체육관광부고시 제2024-27호]를 적용합니다.

### 오픈소스 한글 로마자 변환기 KOROMAN 프로젝트란?

표준 로마자표기법[문화관광부 고시 제2000-8호]에 기반한 온라인 한글-로마자표기법 변환기입니다. 자음동화, 구개음화, 두음법칙 등 한국어 발음 규정을 대부분 준수합니다.

### 오픈소스

- **KOROMAN 프로젝트**: 표준 로마자표기법 기반 온라인 변환기 ([데모 보기](https://daissue.app/romanizer))

# 국어의 로마자 표기법

[시행 2024. 5. 23.] [문화체육관광부고시 제2024-27호, 2024. 5. 23., 타법개정]

문화체육관광부(국어정책과), 044-203-2536

## 제1장 표기의 기본 원칙

- **제1항**: 국어의 로마자 표기는 국어의 표준 발음법에 따라 적는 것을 원칙으로 한다.
- **제2항**: 로마자 이외의 부호는 되도록 사용하지 않는다.
- **제3항**: 삭제

## 제2장 표기 일람

### 제1항 모음

모음은 다음 각호와 같이 적는다.

#### 1. 단모음

| 한글       | ㅏ  | ㅓ  | ㅗ  | ㅜ  | ㅡ  | ㅣ  | ㅐ  | ㅔ  | ㅚ  | ㅟ  |
| :--------- | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
| **로마자** |  a  | eo  |  o  |  u  | eu  |  i  | ae  |  e  | oe  | wi  |

#### 2. 이중 모음

| 한글       | ㅑ  | ㅕ  | ㅛ  | ㅠ  | ㅒ  | ㅖ  | ㅘ  | ㅙ  | ㅝ  | ㅞ  | ㅢ  |
| :--------- | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: | :-: |
| **로마자** | ya  | yeo | yo  | yu  | yae | ye  | wa  | wae | wo  | we  | ui  |

> [!NOTE]
>
> - [붙임 1] ‘ㅢ’는 ‘ㅣ’로 소리 나더라도 **ui**로 적는다. (보기: 광희문 Gwanghuimun)
>
> (보기)

| 한글       |   광희문    |
| :--------- | :---------: |
| **로마자** | Gwanghuimun |

> - [붙임 2] 장모음의 표기는 따로 하지 않는다.

### 제2항 자음

자음은 다음 각호와 같이 적는다.

#### 1. 파열음

| 한글       |  ㄱ  | ㄲ  | ㅋ  |  ㄷ  | ㄸ  | ㅌ  |  ㅂ  | ㅃ  | ㅍ  |
| :--------- | :--: | :-: | :-: | :--: | :-: | :-: | :--: | :-: | :-: |
| **로마자** | g, k | kk  |  k  | d, t | tt  |  t  | b, p | pp  |  p  |

#### 2. 파찰음

| 한글       | ㅈ  | ㅉ  | ㅊ  |
| :--------- | :-: | :-: | :-: |
| **로마자** |  j  | jj  | ch  |

#### 3. 마찰음

| 한글       | ㅅ  | ㅆ  | ㅎ  |
| :--------- | :-: | :-: | :-: |
| **로마자** |  s  | ss  |  h  |

#### 4. 비음

| 한글       | ㄴ  | ㅁ  | ㅇ  |
| :--------- | :-: | :-: | :-: |
| **로마자** |  n  |  m  | ng  |

#### 5. 유음

| 한글       |  ㄹ  |
| :--------- | :--: |
| **로마자** | r, l |

> [!NOTE]
>
> - [붙임 1] ‘ㄱ, ㄷ, ㅂ’은 모음 앞에서는 ‘g, d, b’로, 자음 앞이나 어말에서는 ‘k, t, p’로 적는다.
> - [붙임 2] ‘ㄹ’은 모음 앞에서는 ‘r’로, 자음 앞이나 어말에서는 ‘l’로 적는다. 단, ‘ㄹㄹ’은 ‘ll’로 적는다.

**자음 표기 예시**
| 한글 | 표기 | 한글 | 표기 | 한글 | 표기 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| 구미 | Gumi | 영동 | Yeongdong | 백암 | Baegam |
| 옥천 | Okcheon | 합덕 | Hapdeok | 호법 | Hobeop |
| 월곶 [월곧] | Wolgot | 벚꽃 [벋꼳] | beotkkot | 한밭 [한받] | Hanbat |
| 구리 | Guri | 설악 | Seorak | 칠곡 | Chilgok |
| 임실 | Imsil | 울릉 | Ulleung | 대관령 [대괄령] | Daegwallyeong |

## 제3장 표기상의 유의점

### 제1항 음운 변화

음운 변화가 일어날 때에는 변화의 결과에 따라 적는다.

#### 1. 자음 사이의 동화 작용

| 단어 | 발음   | 표기     | 단어   | 발음     | 표기      |
| :--- | :----- | :------- | :----- | :------- | :-------- |
| 백마 | [뱅마] | Baengma  | 신문로 | [신문노] | Sinmunno  |
| 종로 | [종노] | Jongno   | 왕십리 | [왕심니] | Wangsimni |
| 별내 | [별래] | Byeollae | 신라   | [실라]   | Silla     |

#### 2. 'ㄴ, ㄹ'이 덧나는 경우

| 단어   | 발음     | 표기       | 단어 | 발음   | 표기   |
| :----- | :------- | :--------- | :--- | :----- | :----- |
| 학여울 | [항녀울] | Hangnyeoul | 알약 | [알략] | allyak |

#### 3. 구개음화

| 단어   | 발음     | 표기    | 단어 | 발음   | 표기  |
| :----- | :------- | :------ | :--- | :----- | :---- |
| 해돋이 | [해도지] | haedoji | 같이 | [가치] | gachi |
| 굳히다 | [구치다] | guchida |      |        |       |

#### 4. 거센소리되기 (ㅎ 축약)

- ‘ㄱ, ㄷ, ㅂ, ㅈ’이 ‘ㅎ’과 합하여 거센소리로 소리 나는 경우
- 예시: **좋고**[조코] joko, **놓다**[노타] nota, **잡혀**[자펴] japyeo, **낳지**[나치] nachi
- (보기)

|                   |                  |
| ----------------- | ---------------- |
| 종고[조고] joko   | 놓다[노타] nota  |
| 잡혀[자펴] japyeo | 날지[나치] nachi |

- **다만**, 체언에서 ‘ㄱ, ㄷ, ㅂ’ 뒤에 ‘ㅎ’이 따를 때에는 ‘ㅎ’을 밝혀 적는다.
  - 예시: 묵호(Mukho), 집현전(Jiphyeonjeon)

(보기)

|            |                     |
| ---------- | ------------------- |
| 목호 Mukho | 집현전 Jiphyeonjeon |
| ---------- | ------------------  |

> [!TIP]
> [붙임] 된소리되기는 표기에 반영하지 않는다.
>
> - **압구정** Apgujeong, **낙동강** Nakdonggang, **죽변** Jukbyeon
> - **낙성대** Nakseongdae, **합정** Hapjeong, **팔당** Paldang
> - **샛별** saetbyeol, **울산** Ulsan

(보기)

|                  |                    |
| ---------------- | ------------------ |
| 압구정 Apgujeong | 낙동강 Nakdonggang |
| 죽변 Jukbyeon    | 낙성대 Nakseongdae |
| 합정 Hapjeong    | 팔당 Paldang       |
| 샛별 saetbyeol   | 울산 Ulsan         |

### 제2항 붙임표(-)의 사용

발음상 혼동의 우려가 있을 때에는 음절 사이에 붙임표(-)를 쓸 수 있다.

- 예시: **중앙** Jung-ang, **반구대** Ban-gudae, **세운** Se-un, **해운대** Hae-undae

(보기)

|               |                  |
| ------------- | ---------------- |
| 중앙 Jung-ang | 반구대 Ban-gudae |
| 세운 Se-un    | 해운대 Hae-undae |

### 제3항 고유 명사

고유 명사는 첫 글자를 대문자로 적는다.

- 예시: **부산** Busan, **세종** Sejong

(보기)

|            |             |
| ---------- | ----------- |
| 부산 Busan | 세종 Sejong |
| ---------- | ----------- |

### 제4항 인명

성과 이름의 순서로 띄어 쓴다. 이름은 붙여 쓰는 것을 원칙으로 하되 음절 사이에 붙임표(-)를 쓰는 것을 허용한다.

- 예시: **민용하** Min Yongha (Min Yong-ha), **송나리** Song Nari (Song Na-ri)

(보기)

|                                 |
| ------------------------------- |
| 민용하 Min Yongha (Min Yong-ha) |
| 송나리 Song Nari (Song Na-ri)   |

- **참고**: 이름에서 일어나는 음운 변화는 표기에 반영하지 않는다. (예: 한복남 Han Boknam)

(보기)

|                                 |
| ------------------------------- |
| 한복남 Han Boknam (Han Bok-nam) |
| 홍빛나 Hong Bitna (Hong Bit-na) |

- **참고**: 성의 표기는 따로 정한다.

### 제5항 행정 구역 및 도로명

‘도, 시, 군, 구, 읍, 면, 리, 동, 가’의 단위 앞에는 붙임표(-)를 넣는다. 붙임표 앞뒤에서 일어나는 음운 변화는 표기에 반영하지 않는다.

- 예시: **충청북도** Chungcheongbuk-do, **제주도** Jeju-do, **의정부시** Uijeongbu-si, **양주군** Yangju-gun, **도봉구** Dobong-gu, **신창읍** Sinchang-eup, **삼죽면** Samjuk-myeon, **인왕리** Inwang-ri, **당산동** Dangsan-dong, **봉천 1동** Bongcheon 1(il)-dong, **종로 2가** Jongno 2(i)-ga

(보기)

| 의정부시 Uijeongbu-si      | 양주군 Yangju-gun             |
| 도봉구 Dobong-gu           | 신창읍 Sinchang-eup           |

### 제6항 자연 지물 및 문화재명

자연 지물명, 문화재명, 인공 축조물명은 붙임표(-) 없이 붙여 쓴다.

### 제7항 관용 표기

인명, 회사명, 단체명 등은 그동안 써 온 표기를 쓸 수 있다.

### 제8항 학술 연구 및 특수 분야

글자 대응(transliteration) 시에는 한글 표기를 대상으로 적는다.

- ‘ㄱ, ㄷ, ㅂ, ㄹ’은 ‘g, d, b, l’로만 고정하여 적는다.
- 음가 없는 ‘ㅇ’은 붙임표(-)로 표기(어두 제외)한다. 기타 분절의 필요가 있을 때에도 붙임표(-)를 쓴다.
- 예시: **집** jib, **밖** bakk, **값** gabs, **독립** doglib, **굳이** gud-i, **좋다** johda

(보기)

|        |           |            |                    |
| ------ | --------- | ---------- | ------------------ |
| 집     | jib       | 짙         | jip                |
| 박     | bakk      | 값         | gabs               |
| 붓꽃   | buskkoch  | 먹는       | meogneun           |
| 독립   | doglib    | 문리       | munli              |
| 물엿   | mul-yeos  | 굳이       | gud-i              |
| 좋다   | johda     | 가곡       | gagog              |
| 조랑말 | jolangmal | 없었습니다 | eobs-eoss-seubnida |
